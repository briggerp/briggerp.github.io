MSc Robotics, Systems and Control

I am currently pursuing my MSc in Robotics, Systems and Control at ETH Zurich. This program has equipped me with advanced knowledge and skills in the field of robotics, focusing on various aspects such as autonomous systems, control systems, and human-robot interaction. Below are some of the key courses and projects I have completed during my studies.

National University of Singapore (NUS)

Autonomous Robot Systems

I completed two group projects in this course. In the first project, we navigated a TurtleBot3 robot through a maze with obstacles using Theta* for path planning and bi-directional motion control. The second project involved programming a drone to track the TurtleBot3 in simulation using an Extended Kalman Filter for state estimation and dynamic path planning. These projects enhanced my Python and Matlab skills, ROS capabilities, and practical application of theoretical concepts. See the project reports here: Project 1, Project 2.

In the Adaptive Control Systems course, I implemented adaptive and sliding controllers in Matlab to control a simulated DC motor. See my project reports here: Project 1, Project 2, Project 3.

Control Systems Design and Simulation

ETH Zurich

Master Thesis: Data-Driven Control

My Master's thesis focused on data selection for Data-EnablEd Predictive Control (DeePC) in urban traffic control. DeePC was shown to reduce traffic congestion and emissions more effectively than traditional MPC controllers. However, it is highly sensitive to the data it uses. My thesis aimed to determine which data should be used under varying traffic conditions to ensure the effectiveness of DeePC. This research is part of the REAL initiative of NCCR Automation, where I regularly present my findings to enhance my presentation skills and learn to convey information concisely to diverse audiences. Learn more about the REAL initiative.

Semester Project: Imitation Learning

For my semester project, I developed a neural network to control a motorized labyrinth game by imitating a Model-Predictive Control (MPC) expert. This imitation learning approach allowed for higher bandwidth control with a similar success rate. Through this project, I gained experience with machine learning libraries such as PyTorch and TensorFlow, and furthered my knowledge of ROS2. Read my report for more details: Semester project report.

Physical Human Robot Interaction

In the Physical Human Robot Interaction course, I learned about the safety and technical aspects of human-robot interaction, useful sensors, and control methods that influence interaction experiences. For the final project, a colleague and I programmed a virtual wall with a haptic paddle using an admittance controller. This project enhanced my skills in sensor calibration, LabView programming, and practical application of theoretical knowledge.
